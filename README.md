# UHF_AS_New
新版R2000库使用，读写盘点数据通过回调返回

-  导入依赖库
**AndroidStudio** build.gradle中的dependencies中添加

```
//最外层build.gradle
allprojects {
    repositories {
        jcenter()
        maven { url 'https://jitpack.io' }
    }
}
```
```
 dependencies {
    compile 'com.github.SpeedataG:UHF:7.6.9'
  }
```
